A clean, responsive personal portfolio website for a graphic designer, built with vanilla HTML, CSS, and JavaScript. Features smooth scroll animations, a lightbox image gallery, and mobile-friendly navigation.

---

## 🔗 Live Demo

> [View Portfolio](https://tayyabaebae.github.io/) 

---

## ✨ Features

- **Responsive Design** — Mobile-first layout with a hamburger sidebar for smaller screens
- **Typing Animation** — Dynamic role display in the hero section
- **Scroll Animations** — Powered by [AOS (Animate On Scroll)](https://michalsnik.github.io/aos/)
- **Lightbox Gallery** — Click-to-expand portfolio images using [LightGallery](https://www.lightgalleryjs.com/)
- **Smooth Navigation** — Anchor-based single-page layout with active link highlighting
- **Social Links** — LinkedIn, Instagram, and Behance integration

---

## 🗂️ Sections

| Section | Description |

## 🛠️ Tech Stack

| Technology | Purpose |
|---|---|
| HTML5 | Structure |
| CSS3 | Styling & layout |
| JavaScript (Vanilla) | Sidebar toggle, AOS init, LightGallery init |
| [AOS 2.3.1](https://michalsnik.github.io/aos/) | Scroll-triggered animations |
| [LightGallery 2.7](https://www.lightgalleryjs.com/) | Portfolio image lightbox |
| [Font Awesome 6.5](https://fontawesome.com/) | Icons |
